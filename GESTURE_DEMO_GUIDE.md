## 🎯 手势操作大全

### 基础操作

#### 1️⃣ 缩放控制
```
👊 握拳（所有手指收拢）
   ├─ 效果：图谱缩小，远离视角
   ├─ 手势值：0-15%
   └─ 应用场景：查看全局布局

✋ 张开（五指展开）
   ├─ 效果：图谱放大，靠近视角
   ├─ 手势值：85-100%
   └─ 应用场景：查看节点细节
```

#### 2️⃣ 旋转控制
```
👈 向左划（手掌向左移动）
   ├─ 效果：图谱水平向左旋转
   ├─ 旋转值：rotation.y < 0
   └─ 应用场景：查看左侧隐藏节点

👉 向右划（手掌向右移动）
   ├─ 效果：图谱水平向右旋转
   ├─ 旋转值：rotation.y > 0
   └─ 应用场景：查看右侧隐藏节点

☝️ 向上划（手掌向上移动）
   ├─ 效果：图谱视角向上俯视
   ├─ 旋转值：rotation.x < 0
   └─ 应用场景：从上方俯瞰结构

👇 向下划（手掌向下移动）
   ├─ 效果：图谱视角向下仰视
   ├─ 旋转值：rotation.x > 0
   └─ 应用场景：从下方仰望结构
```

## 🎓 技巧与窍门

### 新手技巧
1. **慢即是快**: 慢速移动更精确
2. **稳定手腕**: 减少不必要的抖动
3. **保持距离**: 30-50cm最佳
4. **充足光线**: 环境光线越好识别越准

### 进阶技巧
1. **肌肉记忆**: 多练习标准动作
2. **预判惯性**: 提前停止手势
3. **分步控制**: 先旋转后缩放
4. **流畅组合**: 一气呵成完成动作

### 演示技巧
1. **开场惊艳**: 从握拳全景开始
2. **节奏控制**: 快慢结合展示

## 🐛 常见问题速查

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 旋转太灵敏 | 灵敏度高 | 降低SENSITIVITY到1.5 |
| 旋转太迟钝 | 灵敏度低 | 提高SENSITIVITY到5.0 |
| 有明显抖动 | 平滑不足 | 增加历史帧数到5 |
| 方向不对 | 映射反了 | 切换rotation的正负号 |
| 相机翻转 | 角度限制松 | 收紧phi范围限制 |
| 手势无响应 | 光线不足 | 改善照明环境 |
| 识别不准 | 距离不当 | 调整到30-50cm |
