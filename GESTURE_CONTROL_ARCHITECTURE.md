## 手势识别算法

```
手部21个关键点编号:
         20(小指尖)
         ↑
    16(无名指尖)
    ↑
12(中指尖)
↑
    8(食指尖)
    ↑
        4(拇指尖)
        ↑
        ↑
        ↑
    0(手腕)

计算公式:
avgDistance = Σ(distance(fingertip[i], wrist)) / 5

其中:
distance(a, b) = √[(a.x - b.x)² + (a.y - b.y)²]

归一化:
openness = (avgDistance - MIN) / (MAX - MIN)
MIN = 0.15 (握拳时的最小距离)
MAX = 0.35 (张开时的最大距离)

平滑:
smoothed = Σ(history[i]) / 5
history = [t-4, t-3, t-2, t-1, t]
```
