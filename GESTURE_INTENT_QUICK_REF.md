**解决**: 智能意图识别 + 死区保护

## 🎮 现在的使用方式

### 纯缩放
```
1. 固定手腕位置
2. 只动手指：握拳↔张开
✅ 只缩放，不旋转
```

### 纯旋转
```
1. 固定手指张开度（保持50%）
2. 只动手腕：左右/上下移动
✅ 只旋转，不缩放
```

### 顺序控制
```
1. 先缩放 → 停顿1秒
2. 再旋转 → 停顿1秒
3. 系统自动切换模式
✅ Console显示: "🎯 Gesture intent switched to: XXX"
```

## 🐛 快速排查

| 问题 | 解决 |
|------|------|
| 缩放时还是会旋转 | 降低OPENNESS_THRESHOLD到0.06 |
| 旋转时还是会缩放 | 降低MOVEMENT_THRESHOLD到0.010 |
| 小动作也触发 | 增大DEAD_ZONE到0.012 |
| 切换太慢 | 减少稳定帧数到2 |
| 切换太快 | 增加稳定帧数到5 |

## 💡 操作技巧

**✅ 好习惯**:
- 缩放时：固定手腕，只动手指
- 旋转时：固定手指，只动手腕
- 切换时：停顿1秒让系统稳定

**❌ 避免**:
- 同时大幅度动手指和手腕
- 快速连续切换缩放和旋转
- 手抖太厉害（改善光线/稳定姿势）
